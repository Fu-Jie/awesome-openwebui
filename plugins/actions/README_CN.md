## 什么是动作插件？

动作插件（Actions）允许您定义可以从聊天中触发的自定义功能。与过滤器不同，动作插件通常用于：

-   生成特定格式的输出（如思维导图、图表等）
-   与外部 API 交互
-   执行数据转换和处理
-   保存或导出内容到文件
-   创建交互式可视化

---

## 如何使用动作插件

1. 将相应的插件文件（`.py`）下载到您的本地计算机
2. 在 OpenWebUI 的管理员设置中，找到"插件"（Plugins）部分
3. 选择"动作"（Actions）类型
4. 上传您下载的文件
5. 刷新页面后，在聊天设置中选择并启用插件

---

## 如何贡献

我们欢迎社区贡献新的动作插件！

1. **Fork** 本仓库
2. 在 `plugins/actions/` 目录下创建新的插件目录
3. 添加您的插件文件（`.py`）
4. 创建完整的中英文文档（`README_CN.md` 和 `README.md`）
5. 更新本 README 文件，将您的插件添加到列表中
6. 创建 **Pull Request**

### 文档要求

每个插件都应该包含：

-   **核心特性**: 插件的主要功能和优势
-   **安装与配置**: 详细的安装和配置步骤
-   **配置参数**: 所有可配置参数的说明
-   **使用方法**: 基本使用示例和最佳实践
-   **故障排除**: 常见问题和解决方案
