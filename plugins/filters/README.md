# Available Filters

This directory contains various filters that can be used to modify the behavior of Open WebUI. Each filter has its own documentation, which can be found in the `docs` subdirectory.

## Filter List

| Filter Name | Description | Documentation |
| :--- | :--- | :--- |
| Async Context Compression | Reduces token consumption in long conversations. | [English](./docs/async_context_compression.md) / [中文](./docs/async_context_compression_cn.md) |

---

> **Note for Developers**: When adding a new filter, please ensure you also add its corresponding documentation in the `docs` directory and update this list.
