# Filters（过滤器）

[English](./README.md) | 中文

过滤器（Filters）在用户输入发送给 LLM 之前对其进行处理和修改。此目录包含可用于扩展 OpenWebUI 功能的各种过滤器。

## 📋 过滤器列表

| 过滤器名称 | 描述 | 文档 |
| :--- | :--- | :--- |
| **异步上下文压缩** | 通过智能摘要和消息压缩，降低长对话的 Token 消耗，同时保持对话连贯性。 | [中文](./async-context-compression/async_context_compression_cn.md) / [English](./async-context-compression/async_context_compression.md) |

## 🚀 快速开始

### 安装过滤器

1. 选择所需的过滤器目录
2. 下载对应的 `.py` 文件到本地
3. 在 OpenWebUI 管理员设置中，找到"Filters"部分
4. 上传 Python 文件
5. 根据过滤器文档配置相关参数
6. 刷新页面，过滤器即可在聊天设置中启用

## 📖 开发指南

添加新过滤器时，请遵循以下步骤：

1. **创建过滤器目录**：在当前目录下创建新文件夹（例如 `my_filter/`）
2. **编写过滤器代码**：创建 `.py` 文件，在注释中清晰说明功能和配置
3. **编写文档**：
   - 创建 `filter_name.md`（英文版）
   - 创建 `filter_name_cn.md`（中文版）
   - 文档应包含功能说明、配置参数、使用示例和故障排除
4. **更新此列表**：在上述表格中添加新过滤器的信息

## ⚙️ 配置最佳实践

- **优先级管理**：为过滤器设置合理的优先级，确保执行顺序正确
- **参数调优**：根据实际需求调整过滤器参数
- **日志调试**：在开发阶段启用 debug 模式，生产环境关闭
- **性能考虑**：测试过滤器在高负载下的表现
